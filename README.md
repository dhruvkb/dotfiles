# .files

**.files** is a collection of my dotfiles, exported settings and other
configuration files, along with documentation on how to use them, published to
GitHub for version control, backup and cross-computer synchronization.

If you see something you like, feel free to use it in your own dotfiles. If you
know of a better way to accomplish something, open an issue.
