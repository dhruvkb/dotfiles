[.files](https://dhruvkb.dev/uses/) is a collection of my dotfiles, exported
settings and other configuration files, along with documentation on
[how to use them](./INSTRUCTIONS.md) to set up a new device, published to GitHub
for the purpose of version control, backup and syncing.

If you see something you like, feel free to use it in your own dotfiles. If you
know of a better way to accomplish something, open an issue.
